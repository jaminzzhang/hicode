# skill-opt outputs

## 定位

本目录用于本地保存 SkillOpt 离线评估训练输出，包括 rollout、checkpoint、日志、模型轨迹和候选 `best_skill.md`。

## Git 边界

除本 `README.md` 外，本目录内容默认不提交到 Git。

需要长期留存的结论，应转写为脱敏摘要或人工采纳记录，放入 `skill-opt/docs/` 下的明确文档中。

## 禁止保存

不得在本目录保存：

1. 密钥、Token、Cookie、Session、连接串或生产凭证。
2. 未脱敏客户信息、真实保单信息、支付信息或生产数据。
3. 生产日志原文、生产配置值、生产接口地址或生产 SQL。
4. 可直接还原真实 MR、真实项目代码或真实会话的原始材料。
